Welcome to the Memory Card Game

An engaging, fun game designed to challenge your memory and concentration skills. This two-player online game allows players to take turns flipping cards and matching pairs of images. The game continues until all pairs are found, and the player with the most card matches wins!

How to Play

  • Setup: The game starts with a shuffled deck of cards laid out face-down. Each card has a matching pair.
  • Turns: Players take turns to flip two cards at a time. If the cards match, they remain face-up, and the player earns a point. If they don't match, the cards are flipped back face-down, and the turn passes to the other player.
  • Winning: The game continues until all cards are matched. The player with the most matched pairs wins the game.

Features

  • Real-time Multiplayer: Play with a friend in real-time. Each player can connect to the game through their browser, and actions are synchronized using WebSockets.
  • Player Information: The game displays each player's score and indicates whose turn it is, ensuring both players are always informed.
  • Visual Feedback: Cards are visually flipped and matched with smooth animations, making the game visually appealing.
  • Restart Option: Players can restart the game to play again.

Technical Details

  • WebSockets: The game uses WebSockets to ensure real-time communication between players. Actions like flipping a card are immediately reflected on both players' screens.
  • Frontend: The game's interface is built using HTML, CSS, and JavaScript. It includes dynamic elements like score updates and turn indicators.
  • Backend: The backend server is implemented using Node.js and the ws library. It handles player connections, game state management, and broadcasts updates to all connected clients.
  • Shuffle Algorithm: The cards are shuffled using a Fisher-Yates shuffle algorithm to ensure a random and fair distribution of card positions.
